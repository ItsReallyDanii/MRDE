# Stage Gate Checklist

## Before moving from Stage 0 to Stage 1

- [ ] `01_MASTER_COMPLETION_CONTRACT.md` exists.
- [ ] `02_STAGE_GATE_PLAN.md` exists.
- [ ] Antigravity rules/skills/workflows exist.
- [ ] `experiments/EXP001/PRE_RUN_PLAN.md` exists as template.
- [ ] `logs/append_only_log.jsonl` exists.
- [ ] No residual data has been inspected.

## Before moving from Stage 1 to Stage 2

- [ ] Pre-run plan is fully filled.
- [ ] Human approval logged.
- [ ] Region/stations/time window locked.
- [ ] Metrics/baselines/thresholds locked.
- [ ] Allowed metadata groupings locked.
- [ ] Sensitivity checks locked.
- [ ] No data inspection occurred before approval.

## Before any claim

- [ ] Validation results exist.
- [ ] Strongest baseline comparison exists.
- [ ] Sensitivity checks exist.
- [ ] Proxy-risk assessment exists.
- [ ] Human review exists.
- [ ] Public language checked.
