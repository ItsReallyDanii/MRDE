# Jules Usage Policy

This document defines the authorized usage scope for the Jules coding agent within the MRDE framework. It strictly controls when the agent may be deployed and what actions it is permitted to take.

## 1. When to Use Jules (Allowed Scopes)

Jules is a highly capable software engineering agent designed to accelerate routine execution and repository maintenance. Jules **may** be used for:

- **Code Engineering:** Writing, refactoring, and optimizing pipeline code (within pre-approved task scopes).
- **Testing:** Writing and running unit tests, integration tests, and schema validation tests.
- **Documentation & Governance:** Drafting documentation updates, formatting markdown files, and preparing stage-gate review documents.
- **Repository Operations:** Summarizing logs, formatting JSON outputs, and preparing Pull Requests (PRs).
- **Tool Development:** Creating local, non-pipeline utility scripts (e.g., in `/home/jules/self_created_tools`) to aid workflow.

## 2. When NOT to Use Jules (Strict Restrictions)

Jules is bound by strict research-control boundaries. Jules **must not** be used for:

- **Residual Preinspection:** Jules must never inspect, summarize, or access residual data unless a human-approved, append-only logged pre-run experiment plan is explicitly confirmed to exist.
- **Scientific Authority:** Jules cannot decide scientific truth, modify experiment hypotheses, or define valid patterns.
- **Claim Upgrades:** Jules is not authorized to execute claim-tier upgrades. Do not use Jules to mark any result as `SUPPORTED`, `VALIDATED`, `PROVEN`, or `DISCOVERED`. Any such language generated must be automatically downgraded to `AMBIGUOUS`.
- **Unauthorized Schema Changes:** Jules must not modify data files or schema contracts without explicit approval and an active task plan.

## 3. Guiding Principles for Agent Interaction

- Treat Jules as an executor, not an arbiter.
- Every run involving Jules must have a clear scope and a pre-defined expected outcome.
- Outputs generated by Jules that touch scientific results must undergo human review before being accepted.
