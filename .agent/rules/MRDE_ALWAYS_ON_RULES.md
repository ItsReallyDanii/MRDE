# MRDE Always-On Agent Rules

## Non-negotiables

1. No residual inspection before an approved pre-run experiment plan exists.
2. No station, metric, threshold, baseline, or scope changes after residual inspection for the same experiment ID.
3. No correction or forecast-improvement claims in MRDE v0.
4. No claim-tier upgrades without human review.
5. No deletion or silent overwriting of failed/ambiguous/rejected results.
6. If governance files conflict, follow the authority order hosted in `README.md`. `README.md` is the reference host for the order, not a governance document that overrides the ordered files.
7. If a task violates these rules, stop and report the blocker.

## Current allowed phase

Stage 0–1 only: governance placement, pre-run planning, templates, schemas, and review.
